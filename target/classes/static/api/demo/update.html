/demo/update.do(案例：修改记录)

只有新建数据库后才能修改记录。

请求地址

环境 HTTP POST 请求地址
正式环境 https://域名/demo/update.do

请求参数

参数 类型 是否必填 最大长度 描述 示例值
token String 必选 50 登录令牌 7752e7b8cff94caf99525a3ad712b82b
DBID String 必选 50 数据库id 5cfe0c881909675da0884e15
id int 必选   id字段值 1
name String 必选 50 name字段值 test

响应参数

参数 类型 是否必填 最大长度 描述 示例值
success boolean 必填 1 true 成功; false 失败 true
code String 必填 4 错误码  0 正确 0
massege String 必填 50 错误信息 成功
result String 选填      

请求示例

  • HTTP请求源码 

        var content = {
                "token": "7752e7b8cff94caf99525a3ad712b82b",
	            "DBID":"5cfe0c881909675da0884e15",
	            "content":{
                    "id":"1",
		            "name":"test1"
	            }
            }
           $.ajax({
               url: "/demo/findAll.do",
               type: "POST",
               contentType: "application/json;charset=UTF-8",
               data: JSON.stringify(content),
               dataType: "json",
               success: function (data) {
                   console.log(data);
                   alert(JSON.stringify(data));
               }
           });

响应示例

  • JSON 示例 
      
{
    "code": 0,
    "success": true,
    "massege": "成功",
    "result": null
}

  • JSON 示例

错误码 错误描述 解决方案

99 未知异常
