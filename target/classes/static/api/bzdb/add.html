/bzdb/add.do(添加业务数据库接口)

只有登录后的用户才允许添加业务数据库。

请求地址

环境 HTTP POST 请求地址
正式环境 https://域名/bzdb/add.do

请求参数

参数 类型 是否必填 最大长度 描述 示例值
token String 必选 256 登录令牌 见代码
company_id String 必选 50 企业id 5d008e2a274e611658ecf55d
name String 必选 50 数据库名称 好友科技

响应参数

参数 类型 是否必填 最大长度 描述 示例值
success boolean 必填 1 true 成功; false 失败 true
code String 必填 4 错误码  0 正确 0
massege String 必填 50 错误信息 成功
result String 选填   返回企业id  

请求示例

  • HTTP请求源码 

        var content = {
               "token":"7752e7b8cff94caf99525a3ad712b82b",
                "content": {
                    "company_id": "5d008e2a274e611658ecf55d",
                    "name": "好友科技"
                }
           }
           $.ajax({
               url: "/company/add.do",
               type: "POST",
               contentType: "application/json;charset=UTF-8",
               data: JSON.stringify(content),
               dataType: "json",
               success: function (data) {
                   console.log(data);
                   alert(JSON.stringify(data));
               }
           });

响应示例

  • JSON 示例 
      
{
	"code": 0,
	"success": true,
	"massege": "成功",
	"result": {
		"id": "5cdd82b61e07f978f8f08fa8"
	}
}

异常示例

  • JSON 示例 

{
	"code": 2,
	"success": false,
	"massege": "没有权限！",
	"result": null
}

错误码

错误码 错误描述 解决方案
1 参数校验错误 检查参数
2 业务数据校验错误 修改业务数据
99 未知异常
